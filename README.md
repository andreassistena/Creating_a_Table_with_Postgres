# Creating_a_Table_with_Postgres
# L1 Exercise 1: Creating a Table with Postgres

## Overview

This exercise demonstrates the fundamental process of creating a table in a PostgreSQL database using SQL. It is part of a larger course module focused on relational database management systems (RDBMS) and emphasizes best practices in schema design and database initialization.

The notebook includes clear, step-by-step code for establishing a database connection, defining a table schema, and executing SQL commands within a Jupyter Notebook environment.

## Tools and Technologies Used

- PostgreSQL
- Jupyter Notebook
- psycopg2 (Python library for PostgreSQL)
- SQL

## Objectives

- Connect to a PostgreSQL database using Python
- Create a table using SQL with appropriate column types
- Understand and implement basic table constraints
- Execute SQL commands within a Python environment

## Files Included

- `L1_Exercise_1_Creating_a_Table_with_Postgres.ipynb`: The primary Jupyter Notebook containing code and markdown explanations
